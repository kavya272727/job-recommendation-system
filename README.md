# Job Recommendation System

## 📌 Overview

This is a simple Python-based job recommendation system that suggests job roles based on user skills using rule-based logic.

## 🚀 Features

* Takes user skills as input
* Matches skills with predefined job roles
* Displays relevant job recommendations

## 🛠️ Technologies Used

* Python
* Basic Data Structures (List, Dictionary)

## ▶️ How to Run

1. Install Python
2. Run the file:
   python job_recommender.py
3. Enter your skills when prompted

## 💡 Future Improvements

* Build a web interface using frameworks
